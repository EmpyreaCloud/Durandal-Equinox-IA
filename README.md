## 📋 Acerca de Durandal Equinox

![Separador-de-dudas-fin-de-temas](https://github.com/user-attachments/assets/4b3a3525-983f-43e5-95d9-bbe7c27140db)


Durandal Equinox emerge de la evolución de nuestro proyecto Karen Proyect IA, representando un significativo paso hacia un gran cambio en Discord. Esta versión trae consigo:

✨ **Características Principales:**
- 🧠 **IA Adaptativa** - Multi-provider (Gemini, Cloudflare, Groq, OpenRouter, Mistral)
- 🎭 **Personalidad Única** - Carga centralizada de `personality.txt` y `faq.txt`
- 🎵 **Sistema Musical Avanzado** - Interfaz gráfica + Lavalink 4.0 + Filtros
- 🎫 **Sistema de Tickets Ultra** - IA integrada + Compras + Alianzas
- 📊 **Server Stats** - Contadores automáticos con temática cyberpunk
- 🛡️ **AutoMod Simplificado** - 1 comando = 1 regla completa
- 📋 **Dashboard Profesional** - Control total desde web (Cloudflare Pages)
- ⚡ **Optimización Extrema** - Sistema propio DurandalOS + DurandalDB

| Comando | Descripción |

| `/shuffle` | Mezclar cola |
| `/loop [modo]` | Cambiar modo loop (off/song/queue) |
| `/filter <tipo>` | Aplicar filtros (bassboost, nightcore, 8d, vaporwave) |
| `/nowplaying` | Canción actual |
| `/musicstatus` | Estado del sistema de música |
| `/node-info` | Información del nodo Lavalink |
